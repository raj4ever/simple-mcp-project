📖 Complete MCP Connection Guide

🚀 Step 1: Start Your MCP Server
# Terminal mein project directory mein jayein
cd /Volumes/Code/mcp

# MCP Server start karein
npm start

# Ya web server start karein
npm run web
🔧 Step 2: Generate API Key

Upar "Generate New API Key" button click karein aur key copy karein.

⚙️ Step 3: Cloud Desktop Configuration

Bas ye simple steps follow karein:

🎯 Simple Setup:
  1. Upar "Generate New API Key" button click karein
  2. "Save to Server" button click karein
  3. Neeche "Copy Ready Config" button click karein
  4. Cloud Desktop mein MCP settings open karein
  5. Copied config paste karein
  6. Cloud Desktop restart karein
📋 Ready-to-Use Configuration:
{
  "mcpServers": {
    "postgresql-mcp-server": {
      "command": "node",
      "args": ["/Volumes/Code/mcp/mcp-server.js"],
      "cwd": "/Volumes/Code/mcp",
      "env": {
        "MCP_API_KEY": "your_api_key_here"
      }
    }
  }
}
🔄 Step 4: Restart Cloud Desktop

Configuration save karne ke baad Cloud Desktop restart karein.

✅ Step 5: Test Connection

Cloud Desktop mein MCP tools check karein:

  • list_users - Users list dekhne ke liye
  • list_products - Products list dekhne ke liye
  • add_user - New user add karne ke liye
  • add_product - New product add karne ke liye
  • update_user - User update karne ke liye
  • update_product - Product update karne ke liye
  • delete_user - User delete karne ke liye
  • delete_product - Product delete karne ke liye
🌐 Step 6: Web Interface Access

Web interface ko access karne ke liye:

http://localhost:8000
🔍 Troubleshooting
❌ Common Issues:
  • Server not found: Check file paths in config
  • Permission denied: Check file permissions
  • API key invalid: Regenerate API key
  • Database locked: Stop other processes using database
🔧 Debug Commands:
# Check if Node.js is installed
node --version

# Check if MCP server is running
ps aux | grep mcp-server

# Check database file
ls -la /Volumes/Code/mcp/database.sqlite

# Test MCP server manually
node /Volumes/Code/mcp/mcp-server.js
